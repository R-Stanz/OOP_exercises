In this exercises you should create and manipulate an
array list. In order to use to use it you must import
it as " java.util.ArrayList ".

To see what are the array list methods check the
documentation: 
https://docs.oracle.com/javase/8/docs/api/java/util/ArrayList.html

Now it's required to you to make a program capable of
storing 5 integers. Print the middle value in one line 
and then print all the remaining values of the array 
list.
